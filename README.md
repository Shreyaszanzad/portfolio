# shreyaszanzad.dev

Personal portfolio — coming soon at [shreyaszanzad.dev](https://shreyaszanzad.dev).

## Tech Stack

- **Framework** — Next.js 16 (App Router)
- **Language** — TypeScript
- **Styling** — Tailwind CSS v4
- **3D / WebGL** — Three.js, React Three Fiber, Drei, Rapier physics
- **Animation** — GSAP (ScrollTrigger), Framer Motion, Lenis smooth scroll
- **UI** — Lucide React, React Icons

## Running Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
